# Основные настройки

В программе используются следующие настройки:

```text
NGLDM_D = 2
N_LEVELS = 16
ALPHA = 1
GAMMA = 0.7
USE_LOG_FOR_VIS = True
```

### NGLDM_D

Радиус окрестности для построения матрицы NGLDM.

Для варианта 9 используется:

```text
d = 2
```

Это означает, что для каждого пикселя анализируется квадратная окрестность размером:

```text
(2d + 1) × (2d + 1) = 5 × 5
```

Максимальный размер зависимости:

```text
25
```

### N_LEVELS

Количество уровней серого после квантования изображения.

В работе используется:

```text
N_LEVELS = 16
```

То есть яркости диапазона `0..255` переводятся в уровни:

```text
0..15
```

### ALPHA

Порог похожести соседних пикселей при построении NGLDM.

В работе используется:

```text
ALPHA = 1
```

Сосед считается похожим на центральный пиксель, если выполняется условие:

```text
|neighbor - center| <= ALPHA
```

### GAMMA

Параметр степенного преобразования яркости.

В работе используется:

```text
GAMMA = 0.7
```

Если `gamma < 1`, изображение становится светлее.  
Если `gamma > 1`, изображение становится темнее.

### USE_LOG_FOR_VIS

Используется для визуализации матрицы NGLDM.

```text
USE_LOG_FOR_VIS = True
```

При включённом параметре применяется логарифмическое нормирование:

```text
log(1 + value)
```

Это нужно, чтобы матрица не выглядела полностью чёрной при больших различиях между значениями.

# Работа с цветовой моделью HSL

В лабораторной работе рекомендуется использовать модель **HSL**, чтобы работать только с яркостным каналом `L` и не менять цветовые компоненты `H` и `S`.

В программе выполняется преобразование:

```text
RGB → HSL
```

Затем отдельно обрабатывается канал яркости:

```text
L
```

После преобразования яркости изображение возвращается обратно:

```text
HSL → RGB
```

Благодаря этому контрастирование влияет на яркость изображения, но не искажает оттенок и насыщенность.

# Степенное преобразование яркости

Для варианта 9 используется степенное преобразование:

```text
L_after = L_before ^ gamma
```

где:

- `L_before` — исходная яркость пикселя в диапазоне `[0, 1]`;
- `gamma` — параметр преобразования;
- `L_after` — яркость после преобразования.

В коде это реализовано функцией:

```python
def power_contrast_channel(channel: np.ndarray, gamma: float = GAMMA):
    out = np.power(np.clip(channel, 0.0, 1.0), gamma)
    out = np.clip(out, 0.0, 1.0)
    return out, fmin, fmax
```

Для выбранного значения:

```text
GAMMA = 0.7
```

изображение становится светлее, а детали в тёмных областях становятся заметнее.

# Матрица NGLDM

**NGLDM** — матрица зависимости близких уровней серого.

Для каждого пикселя выполняются следующие шаги:

1. пиксель переводится в квантованный уровень серого `i`;
2. рассматривается окрестность радиуса `d`;
3. подсчитывается количество соседей, яркость которых близка к яркости центрального пикселя;
4. размер зависимости обозначается как `j`;
5. значение матрицы `NGLDM[i, j]` увеличивается на 1.

В программе размер зависимости считается так:

```text
j = 1 + количество похожих соседей
```

При `d = 2` максимальное значение `j` равно 25.

# Признаки NGLDM

Для варианта 9 рассчитываются признаки:

```text
SNE
LNE
```

## SNE — Small Number Emphasis

Признак `SNE` показывает выраженность малых зависимостей.

Формула:

```text
SNE = Σ P(i, j) / j² / N
```

где:

- `P(i, j)` — элемент матрицы NGLDM;
- `j` — размер зависимости;
- `N` — сумма всех элементов матрицы.

Если `SNE` большое, в изображении много небольших локальных зависимостей. Это может говорить о более мелкой или неоднородной текстуре.

## LNE — Large Number Emphasis

Признак `LNE` показывает выраженность больших зависимостей.

Формула:

```text
LNE = Σ P(i, j) · j² / N
```

Если `LNE` большое, в изображении много крупных однородных областей, где соседние пиксели похожи друг на друга.

# Как интерпретировать результат

После запуска программы нужно сравнить значения признаков до и после преобразования.

## Если `SNE` увеличилось

Это значит, что стало больше малых зависимостей. Изображение может выглядеть более детализированным или менее однородным.

## Если `SNE` уменьшилось

Это значит, что мелких локальных зависимостей стало меньше.

## Если `LNE` увеличилось

Это значит, что стало больше крупных однородных областей.

## Если `LNE` уменьшилось

Это значит, что крупные однородные области стали менее выраженными.

Степенное преобразование яркости меняет распределение яркости, поэтому после него могут измениться квантованные уровни серого, матрица NGLDM и рассчитанные признаки.
